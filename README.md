# 👁️ Visionyx

Visionyx is a real-time computer vision system that performs face tracking, eye detection, and emotion + age estimation with a futuristic AR-style HUD interface.

It overlays live analytics directly on webcam input, creating a lightweight “AI vision dashboard” experience.

---

## 🚀 Features

- 👤 Real-time face detection & tracking  
- 👁️ Eye detection overlay  
- 😊 Emotion recognition  
- 🧠 Age group estimation  
- 🧊 Glass-style HUD interface  
- 🔷 Face mesh visualization (grid-style landmarks)  
- 🎯 Face focus ring (AR-style targeting system)  
- ⚡ Lightweight OpenCV-based rendering
